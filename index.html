About

3D Visual Graph is a WebGL graph and cluster viewer, built with Babylon.js.
This application is used for viewing three-dimensional graphs and clusters using the default file listed below:

  • [.S3D] 3D Graph generated using Kruskal Algorithm;
  • [.S3D] 3D Cluster generated using K-Means Algorithm;
  • [.S3D] 3D Cluster generated using J-Means Algorithm.

  • [.XML] 3D Graph generated using Kruskal Algorithm;
  • [.XML] 3D Graph using customized path;
  • [.XML] 3D Cluster generated using K-Means Algorithm.
Every .s3d file follows the same pattern, using the first three coordinates (x, y, z) as the structure vertex, the next three coordinates as its centroid (for clusters) and its connected vertex (for graphs). The last three data (red, green, blue) are the color used on the structure.
The .xml file follows a different pattern, where all the vertex are defined at the <VERTICES> tag, using the three coordinates (x, y, z), an ID for each one, and its color. The edges are defined at the <LINKS> tag, and uses the vertex ID to draw the edge between the two vertex, and uses its color.
